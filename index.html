Basic Steps of Shopify Theme Development

1. Installation

First, install the Shopify CLI or Theme Kit. We recommend using the Shopify CLI. To install, follow these steps:

npm install -g @shopify/cli
npm install -g @shopify/theme

2. Connect to Live Theme

Connect your local development environment to a live theme in Shopify using Visual Studio Code (VS Code). Here's how:

  1. Create an empty theme in your Shopify store and activate it.
  2. Pull the theme from Shopify to your local machine by running the command:
shopify theme pull

3. Run the Development Server

Run the development server in VS Code by executing the command:

shopify theme dev

4. Monitor Code Changes

Monitor your code changes and detect any errors by running:

shopify theme serve

5. Open in VS Code

Open the project folder in VS Code using the shortcut command:

code .

Troubleshooting in the CLI

If you encounter any errors or problems, you can uninstall and reinstall the CLI. Use the following commands:

npm uninstall -g @shopify/cli
npm uninstall -g @shopify/theme

Software Requirements

  1. Node.js
  2. Ruby with DevKit
  3. Git
  4. VS Code or Sublime Text (VS Code is recommended). If you're using VS Code, install the "liquid" extensions from the marketplace. Choose 3-4 extensions from the list.
